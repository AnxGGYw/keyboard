# vim3

操作符 （operator） + 动作 (范围)

## practice

```js
function getName() {
  return 'aaaa-bbbb-cccc'
}
```

> - 删除 -> d
> - 删除并进入 insert 模式 -> c
> - 复制 -> y

> - 移动到单词结尾 -> e
> - 移动到单词开头 -> w
> - 移动到上一个单词结尾 -> ge
> - 移动到上一个单词开头 -> b

### 组合

> - 删除当前单词 -> dw / cw
> - 在当前单词尾部添加 -> ea
> - 在当前单词头部添加 -> bi
